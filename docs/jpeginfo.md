"jpeginfo - prints information and tests integrity of JPEG/JFIF files." - man jpeginfo

# Example

## Test integrity of all files in a dir

```
jpeginfo -c *
```

## Test integrity of files and delete any corrupt files

```
jpeginfo -c -d *
```

# DevOps lifecycle

- Plan - Jira
- Code - Git, Eclipse, pycharm
- Build - Ant, Maven, Gradle
- Test - Selenium, JUnit
- Release - Jenkins, Bamboo
- Deploy - Puppet, Chef, Ansible, Saltstack
- Operate - Linux, Chrome, iOS
- Monitor - Sensu, Splunk, Nagios

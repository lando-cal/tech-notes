## apg

This tool lets you show how to pronounce the random password

```
$ apg -a 1 -m 6 -n 3 -l

WoqrJ}R+ps Whiskey-oscar-quebec-romeo-Juliett-RIGHT_BRACE-Romeo-PLUS_SIGN-papa-sierra

```
